# HTML CSS JS Projects

This collection features various frontend web development mini-projects demonstrating HTML, CSS, and JavaScript techniques.

**[View Live Demo](https://fontend-mini-projects.netlify.app/)**

## Projects List

- **[3D Image Slider](3d%20Image%20Slider%20with%20Carousel%20using%20HTML.html)**([Live Demo](https://fontend-mini-projects.netlify.app/3d%20Image%20Slider%20with%20Carousel%20using%20HTML.html  )): A 3D rotating image carousel built using CSS 3D transforms.
- **[Days Between Two Dates](Days%20Between%20Two%20Dates%20Using%20JavaScript.html)** ([Live Demo  ](https://fontend-mini-projects.netlify.app/days%20between%20two%20dates%20using%20javascript.html  )): A utility to calculate the number of days between a start and end date using JavaScript.
- **[Glassmorphism Profile Card](Glassmorphism%20Profile%20Card%20Design%20using%20HTML%20and%20CSS.html)** ([Live Demo](https://fontend-mini-projects.netlify.app/glassmorphism%20profile%20card%20design%20using%20html%20and%20css.html)): A modern UI component showcasing the glassmorphism effect with backdrop filters.
- **[Login and Registration Form](Login%20and%20Registration%20Form%20using%20HTML%20and%20CSS.html)** ([Live Demo](https://fontend-mini-projects.netlify.app/login%20and%20registration%20form%20using%20html%20and%20css)): A sliding form interface that toggles between login and registration views.
- **[Random Joke Generator](Random%20Joke%20Generatorhtml%20css.html)** ([Live Demo](https://fontend-mini-projects.netlify.app/random%20joke%20generatorhtml%20css.html )): A web app that fetches data from an API to display random jokes.
- **[Team Section](Team%20Section%20html%20css.html)** ([Live Demo ](https://fontend-mini-projects.netlify.app/team%20section%20html%20css.html )): A responsive layout for displaying team member profiles with hover animations.

## Usage

1. Open `index.html` in your web browser to see the main directory.
2. Click on any project card to view the live demo.
3. Alternatively, you can open the individual `.html` files directly.
